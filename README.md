# 💳 Credit Scoring Model using Machine Learning

## 📌 Project Overview

Financial institutions use credit scoring models to determine whether a customer is likely to repay a loan. This project applies Machine Learning classification algorithms to predict an individual's creditworthiness based on historical financial information.

The project compares multiple classification algorithms and evaluates their performance using industry-standard metrics.

---

## 🎯 Objective

Predict whether a customer is creditworthy using financial history such as:

- Annual Income
- Monthly Debt
- Loan Amount
- Payment History
- Credit Utilization
- Number of Credit Cards
- Previous Defaults
- Employment Status
- Credit History Length

## 📂 Dataset Features

Example attributes include:

| Feature | Description |
|----------|-------------|
| Age | Customer Age |
| Annual Income | Total yearly income |
| Monthly Debt | Monthly debt payments |
| Credit Utilization | Percentage of credit used |
| Number of Loans | Existing loans |
| Payment History | On-time payment percentage |
| Credit History Length | Years of credit history |
| Employment Status | Employed / Self-employed |
| Previous Defaults | Number of previous defaults |
| Target | Creditworthy (Yes/No) |
